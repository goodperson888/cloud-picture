POST
/api/auth/register
B端用户自主注册
B端用户自主注册接口，需要提供有效的销售推荐码和手机验证码。注册成功后系统会自动创建推荐关系记录，并返回登录令牌供直接访问系统。用户名要求3-50个字符，密码至少6位，手机号必须为中国大陆三大运营商号段，推荐码必须存在且处于激活状态。

Parameters
Try it out
Name	Description
request *
object
(body)
用户注册请求体，包含用户名、邮箱、手机号、密码、验证码、公司名称和必需的推荐码

Example Value
Model
{
  "captcha": "1111",
  "company_name": "测试科技有限公司",
  "email": "test@example.com",
  "password": "password123",
  "phone": "13800138000",
  "referral_code": "SALES001",
  "username": "testuser"
}
Parameter content type

application/json
Responses
Response content type

application/json
Code	Description
200	
注册成功，返回用户信息和访问令牌，可直接用于系统访问

Example Value
Model
{
  "code": 0,
  "data": {
    "access_token": "string",
    "expires_in": 0,

    "user": "string"
  },
  "message": "string"
}

请求参数错误，包括：用户名长度不符合要求、邮箱格式错误、手机号格式错误、密码过短、验证码错误、推荐码无效或已失效等

  "code": 0,
  "data": "string",
  "message": "string"
}
409	
用户已存在，用户名、邮箱或手机号已被注册，请使用其他信息

  "data": "string",
  "message": "string"
}
500	
服务器内部错误，注册过程中发生异常

Example Value

POST
/api/auth/send-captcha
发送验证码
向指定手机号发送验证码，支持注册、登录、重置密码等场景。限制：每个IP每分钟最多3次请求，每个手机号每分钟最多1次请求。

object
